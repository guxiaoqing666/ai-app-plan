# 隐私政策简版模板

本产品为 Citywalk 路线推荐工具。首版不要求注册账号，不收集身份证、精确定位、通讯录、相册等敏感信息。

## 我们可能收集的信息

- 用户主动填写的联系方式，如微信、邮箱或手机号。
- 用户主动提交的反馈内容。
- 用户选择的城市、预算、同行类型和路线偏好。
- 基础访问和使用数据，如路线查看、复制、反馈提交。

## 信息用途

- 改进路线推荐质量。
- 联系愿意参与测试或预约高级版的用户。
- 判断是否开发实时 AI、导出图片、付费功能等后续能力。

## 信息存储

首版静态版本会把反馈保存在当前浏览器。正式上线接入外部表单或数据库后，应在页面中更新具体的数据存储位置和删除方式。

## 用户权利

用户可以要求删除自己主动提交的联系方式和反馈。正式上线后，应提供一个明确的联系邮箱或微信。

## 免责声明

路线中的营业时间、价格、交通和地点信息可能变化。出行前请以地图、商家和现场信息为准。
